警报是一个彩色文本框。

## 1. 如何使用

```
{{</* alert theme="warning" dir="ltr" */>}} # warning, success, info, danger
**this** is a text
{{</* /alert */>}}
```

## 2. 长什么样子

{{< alert theme="warning" dir="ltr" >}}
**this** is a text
{{< /alert >}}

{{< alert theme="success" dir="ltr" >}}
**this** is a text
{{< /alert >}}

{{< alert theme="info" dir="ltr" >}}
**this** is a text
{{< /alert >}}

{{< alert theme="danger" dir="ltr" >}}
**this** is a text
{{< /alert >}}
