## 1. How to use

- Markdownify box

    ```
    {{</* boxmd */>}}
    Some markdown contents
    {{</* /boxmd */>}}
    ```

- Simple box

    ```
    {{</* box */>}}
    Some contents
    {{</* /box */>}}
    ```
